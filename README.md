# Financial Modelling Project

This project performs financial analysis on banking datasets using Python and PostgreSQL.

## Features
- CSV Data Analysis
- ROI Calculation
- Growth Rate Analysis
- Excel Report Generation
- Financial Data Processing

## Technologies Used
- Python
- Pandas
- NumPy
- PostgreSQL
- DBeaver
- PyCharm

## Projects Included
- Union Bank Financial Modelling
- Central Bank Financial Modelling

## Output
The project generates automated Excel reports containing:
- Profit Analysis
- Growth Rate
- ROI
- Financial Summary
